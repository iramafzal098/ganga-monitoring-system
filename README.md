# 🌊 Ganga Monitoring System

An AI-powered river water quality monitoring and analytics dashboard for the Ganga River.

## 🚀 Features

- 📍 Interactive pollution intelligence map
- 📊 Water quality analytics dashboard
- 🤖 AI pollution forecasting
- 🚨 Real-time alert system
- 👤 User registration & profile management
- 🌐 Responsive modern UI
- 📈 Dynamic charts using Chart.js
- 🗺️ Live map visualization using Leaflet.js

---
## 🧠 Problem Statement

The Ganga River faces severe pollution challenges due to:
- industrial discharge
- sewage contamination
- declining dissolved oxygen
- bacterial contamination

This project helps monitor water quality in real-time and provides AI-driven insights for early pollution detection.
